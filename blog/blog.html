How I created this Web Portfolio

It took me about five hours to work on the portfolio you are viewing now. First, I created the folders and files in them, then in each file I wrote the head and necessary links such as Bootstrap, CSS and Google Fonts. When I finish with Navbar, I decided to also attach a footer where I placed a link to GitHub and Netlify. After that, all I had to do - was fill the files with content, add media, and adjust the styles. To be honest, I decided on a writing style the longest time, but I like this part of the job because that's how I can express my creativity.
